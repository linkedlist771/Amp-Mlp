### 上手指南

###### 开发前的配置要求

1. python >= 3.10
2. PyTorch >= 2.1.0
3. pandas

###### 安装步骤

1. 克隆仓库

```sh
git clone https://github.com/linkedlist771/Amp-Mlp.git
```

2. 安装依赖

```bash
pip install -r requirements.txt
```
